Welcome to Matt Harvey Online

Build The Flock Online: A Twitter Networking Front End Project

Build The Flock is one of my first front end projects and serves as a web hub for a social media networking campaign to facilitate bringing the Louisville athletics community on twitter more in touch with each other. Before I built the site I was doing mass twitter shoutouts regularly to try to get U of L fans to keep up with each other online. The site made it easier for me to try to promote people following each other and helped a lot of people build their social media presence. This will likely always be a work in progress for me but it was a nice way for me to get my feet wet as I was exploring front end design.

Louisville Crime Analysis with Python

This project reads in a selection of data from Louisville Metro Government on crime within the city into an SQLite database that I created and organized. It then queries that database for custom selection of data which it organizes a plethora of different ways including as a text chart, pie chart, and bar chart. Creating custom visualizations of a data set was a pleasure for me and opened my eyes to a lot of possibilities, creating a curiosity about the further possibilities with Pandas and MatPlotLib as well as what else may be out there to investigate data further.

Querying UFO Sightings with C#/.NET

This project takes a .csv file obtained from Kaggle and originally compiled by NASA of reported UFO sightings domestic and international and queries it to create a custom result. This app runs in a console and makes heavy use of system.linq to allow you to search the data by city, state, country, or date and create a data set of your specification. It allows for all four CRUD operations create, read, update and delete to allow a comprehensive functionality for data manipulation.

Meet The Flockers Tailgate Meetup React.js App

Meet The Flockers is a React.js web app serving as a tool to facilitate tailgate networking. Meet The Flockers calls 3 apis for data on weather, game day statistics and a google map of the area surrounding the stadium for each game, home and away. This allows for people to highlight where they will be tailgating as well as providing weather info for pre game preparation and statistical information to provide talking points to discuss while meeting people in the tailgate parking lots.
